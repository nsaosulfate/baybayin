> **✅ 확정 사항 (2026-06-14)**
> - 앱 이름: **Baybay**
> - v1 범위: **게스트 모드(로컬 저장) 먼저** — 로그인/클라우드는 나중에

## 2. ⚠️ 가나와 다른 결정적 포인트 — 폰트

가나(히라가나·가타카나)는 아이폰·윈도우에 **기본 폰트가 내장**돼 있어서 그냥 글자만 넣으면 보였다.
하지만 **바이바이인 글자(ᜃ, ᜊ, ᜈ …)는 대부분 기기에서 기본으로 안 보인다**(빈 네모로 뜸).

→ **해결:** 구글 폰트 **Noto Sans Tagalog** 를 웹폰트로 임베드한다(인터넷 연결 시 자동 로드).
가나 앱에서 `--kana-font` 변수 쓰던 자리를 `--baybayin-font` 로 바꿔서 동일하게 처리.

```html
<link href="https://fonts.googleapis.com/css2?family=Noto+Sans+Tagalog&display=swap" rel="stylesheet">
```
```css
--baybayin-font: "Noto Sans Tagalog", sans-serif;
```

## 3. 글자 데이터 (Baybayin 문자 체계)

바이바이인은 **아부기다**(자음에 기본 모음 'a'가 붙어있는 음절문자)다.
가나의 `히라/가타 + 탁음/요음` 파트 구조를 → **모음 / 기본 / 쿠들릿i / 쿠들릿u / (비라마)** 로 매핑한다.

### 파트 ① 모음 (Patinig) — 3자
| 글자 | 로마자 | 한글 |
|---|---|---|
| ᜀ | a | 아 |
| ᜁ | i / e | 이 (에) |
| ᜂ | u / o | 우 (오) |

### 파트 ② 기본 자음 +a (Katinig, 기본형) — 14자
| 글자 | 로마자 | 한글 | | 글자 | 로마자 | 한글 |
|---|---|---|---|---|---|---|
| ᜃ | ka | 카 | | ᜋ | ma | 마 |
| ᜄ | ga | 가 | | ᜌ | ya | 야 |
| ᜅ | nga | 응아 | | ᜎ | la | 라 |
| ᜆ | ta | 타 | | ᜏ | wa | 와 |
| ᜇ | da / ra | 다 (라) | | ᜐ | sa | 사 |
| ᜈ | na | 나 | | ᜑ | ha | 하 |
| ᜉ | pa | 파 | | | | |
| ᜊ | ba | 바 | | | | |

### 파트 ③ 쿠들릿 i/e (위에 점) — 14자
ᜃᜒ ki · ᜄᜒ gi · ᜅᜒ ngi · ᜆᜒ ti · ᜇᜒ di/ri · ᜈᜒ ni · ᜉᜒ pi · ᜊᜒ bi · ᜋᜒ mi · ᜌᜒ yi · ᜎᜒ li · ᜏᜒ wi · ᜐᜒ si · ᜑᜒ hi

### 파트 ④ 쿠들릿 u/o (아래 점) — 14자
ᜃᜓ ku · ᜄᜓ gu · ᜅᜓ ngu · ᜆᜓ tu · ᜇᜓ du/ru · ᜈᜓ nu · ᜉᜓ pu · ᜊᜓ bu · ᜋᜓ mu · ᜌᜓ yu · ᜎᜓ lu · ᜏᜓ wu · ᜐᜓ su · ᜑᜓ hu

### 파트 ⑤ 비라마 = 받침/순수 자음 (크루스 쿠들릿 ᜔) — 14자 ✅ 포함
ᜃ᜔ k · ᜄ᜔ g · ᜅ᜔ ng · ᜆ᜔ t · ᜇ᜔ d/r · ᜈ᜔ n · ᜉ᜔ p · ᜊ᜔ b · ᜋ᜔ m · ᜌ᜔ y · ᜎ᜔ l · ᜏ᜔ w · ᜐ᜔ s · ᜑ᜔ h
(스페인 식민기에 추가된 받침 표기)

> **합계:** **59자** (모음 3 + 기본 14 + 쿠들릿i 14 + 쿠들릿u 14 + 비라마 14).
> 가나(약 200자)보다 적어서 학습 난이도·코드량 모두 가볍다.
> 데이터 생성도 Kanade의 `mk()` 패턴 그대로: `{char, roma, kor, note, cat}` 배열.

## 4. 발음 (TTS)

- 가나: Web Speech API `ja-JP` 로 글자 탭 시 발음 재생.
- Baybayin: 음절 로마자("ka","ki","ku"…)를 **`fil-PH`(필리핀어)** 로 읽어준다.
- 코드 변경: `pickJaVoice` → `pickFilVoice`, `u.lang = "fil-PH"` 정도만.
- ⚠️ 주의: 아이폰에 필리핀어 음성이 없을 수 있음 → 없으면 영어 음성으로 폴백하거나 음성 기능 생략.
  (기기에서 실제 확인 필요. 가나 때도 프리뷰는 무음이고 아이폰에서 확인했음.)

- **앱 이름: Baybay** (확정) — "철자하다/표기하다"라는 뜻, Baybayin과 직관적으로 연결됨

## 6. 작업 단계 (Phase)

1. **복제 & 정리** — `가나_암기카드.html` 복사 → `Baybayin/` 에 새 HTML로. 가나 데이터/문구 제거.
2. **데이터 입력** — 위 3절 표대로 POOLS 작성 (모음/기본/쿠들릿i/쿠들릿u).
3. **폰트 적용** — Noto Sans Tagalog 임베드, 표·카드 글자 렌더 확인(빈 네모 안 뜨게).
4. **발음** — TTS `fil-PH` 연결.
5. **브랜딩** — 이름/아이콘/타이틀/색상.
6. **로컬 검증** — 프리뷰로 화면·다크모드·한영전환·퀴즈 동작 확인.
7. **(원하면) 클라우드** — 새 Firebase 프로젝트 + 새 GitHub Pages 저장소로 배포 + 로그인.

> v1은 **1~6단계(게스트 모드 + 로컬 저장)** 까지만 해도 완전히 동작.
> 로그인·클라우드 동기화(7단계)는 원할 때 추가하면 된다.
